# Cervical_Fracture_Detection_Using_ML
Normal CT scans often fail to detect occult fractures in cervical bones (C5, C6, C7), leading to prolonged undiagnosed conditions that may cause immobility and spine-related complications. My ML-based model leverages Edge Detection using the Sobel operator to accurately identify fractures, ensuring early diagnosis and timely intervention. This technology has the potential to revolutionize spinal injury detection and improve patient outcomes! 🔍💡
